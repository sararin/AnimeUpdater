```
>>> anup -h
usage: anup [-hl] -s DIR -r ANIME

Updates anime you've watched, removed episodes will mark as seen. Ran without
arguments it will update already watched chineese cartoons. Empty and removed
folders will be marked as "watched"

optional arguments:
   -h, --help               shows this exit message and exits
   -s DIR, --set DIR        sets anime folder
   -r ANIME, --remove ANIME removes the show and adds it to dropped 
   -l, --list               lists watching/watched/dropped anime
```
